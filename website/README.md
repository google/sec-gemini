# Sec-Gemini Website

This directory contains the source code for the public-facing documentation website for Sec-Gemini.

For the live documentation, please visit: [https://docs.secgemini.google/](https://docs.secgemini.google/)

## Setup & Development

To install the main dependencies, run:

```bash
npm install
```

To run the development server, run:

```bash
npm run dev
```
