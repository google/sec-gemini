Welcome to the documentation for **Sec-Gemini v3**.

Sec-Gemini is an experimental AI agent from Google designed for technical security workflows. It combines advanced reasoning models with agentic tool execution capabilities, allowing security engineers and researchers to automate complex tasks, interact dynamically with local environments, and script AI-driven workflows.

:::note
If you are familiar with Sec-Gemini v1 or v2, please note that Sec-Gemini v3 is a completely overhauled system with a brand new architecture, capabilities, and developer experience.
:::

## Highlights
Sec-Gemini v3 is packed with new features to streamline your cybersecurity workflows:

- **TUI (Terminal User Interface)**: Interact with Sec-Gemini directly from your terminal for a fast, keyboard-centric experience.
- **Python SDK**: A high-level, asyncio-native Python client for easily integrating Sec-Gemini into your applications and automation scripts.
- **Bring Your Own Tools (BYOT)**: Securely connect Sec-Gemini to your local environment or segregated networks, allowing Sec-Gemini to interact with internal logs, compromised machines, or hardware devices.
- **MCP Support**: Use the Model Context Protocol (MCP) to plug in custom tools and data sources.
- **Skills**: Extend Sec-Gemini's capabilities with specialized skills tailored for specific security workflows.

Check out the rest of the documentation to explore these features in detail.
